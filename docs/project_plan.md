# Project Plan

## 1. Choose Domain

Select one domain from the assessment brief:

- Forestry
- Medical imaging
- Agriculture health
- Animal subspecies
- Plant subspecies

Recommended option: agriculture health or plant subspecies because public datasets are easier to find and explain.

## 2. Prepare Dataset

- Collect images.
- Use at least 3 classes.
- Standardize image format and size.
- Remove corrupted or duplicate images.
- Split into training, validation, and testing folders.
- Visualize class distribution.

## 3. Train Models

Train these Keras CNN models using transfer learning:

- ResNet50
- DenseNet121
- MobileNetV3

Each model should be trained for 50 epochs.

## 4. Evaluate Models

For each model, record:

- Accuracy
- Mean average precision
- Training time
- Number of parameters
- Loss graph
- Accuracy graph
- Confusion matrix

## 5. Compare And Conclude

Compare the three models and decide which one is best for the classification task.

The conclusion should consider performance and efficiency, not accuracy alone.
